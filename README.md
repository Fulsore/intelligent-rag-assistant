# RAG Chatbot

A Retrieval-Augmented Generation (RAG) chatbot that allows users to upload documents and receive context-aware responses using Large Language Models.

## Features

- PDF Upload
- Document Processing
- Embedding Generation
- Vector Search
- Semantic Retrieval
- Context-Aware Responses
- LLM Integration
- Conversational Interface

## Tech Stack

### Frontend
- React.js
- Tailwind CSS

### Backend
- Django REST Framework
- Python

### AI Stack
- LangChain
- OpenRouter
- Groq
- Hugging Face

### Vector Database
- ChromaDB / FAISS
